# 🛡️ ChamaSmart v1.1
> **The Fintech Vault with an AI Soul.** > Built for speed, accuracy, and just enough sarcasm.

ChamaSmart is a high-performance, non-custodial ledger system designed for Kenyan Chamas. It bypasses the "Streamlit lag" by using a FastAPI + Vercel backbone to provide instant financial insights on any device—from a high-end EliteBook to a Vivo y03.

---

## ✨ Features
- **🚀 Lightning Fast:** Vercel-hosted backend for sub-second loading in Nairobi.
- **🤖 T.E.S.S. Integration:** Built-in AI humor module that comments on your transactions.
- **🔐 Ironclad Security:** Row-Level Security (RLS) powered by Supabase to keep the "chama" funds private.
- **📱 Mobile Optimized:** Designed to look and feel like a native app on Safaricom's network.

---

## 🛠️ Tech Stack
- **Backend:** Python (FastAPI)
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel + GitHub
- **Styling:** Tailwind CSS (Modern, sleek UI)
